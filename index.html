about me

Android and web developer.

I'm Tomáš Tomovčík. I started playing around with computers waaay back in who knows when. I created my first ever application back in 2009 - a calculator in C# and WinForms, and after that, nothing really happened.

In 2012 I started to *actually code something*. After downloading few sample projects and trying to figure out how things work I gave up realized that this appoach wouldn't work for me. So, being me, I started tinkering with the code and finally figured out what's gonna work for me. The Code as you go approach seemed like a good idea - and it was for a while. But few years later I came to a conclusion that I missed A LOT of coding basics. So I started to read books about coding and learn the basics.

Today, after slowly learning basics I work on Android apps and pretty much everything around AOSP, build (and sometimes also design) websites and experiment with this next-gen machine learning stuff that everyone is talking about.

I currently work as a technician and Android developer, but I'm open for new opportunities. Feel free to contact me. :-)
